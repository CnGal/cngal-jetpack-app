### 背景

我们之前通过 Blazor Hybrid 和 .NET MAUI 共用代码库来生成 App。但是在接入Identity Server 之后鉴权流程发生了巨大更改，MAUI无法继续共用部分鉴权逻辑

加上MAUI本身的种种原因，我们决定使用 Jetpack Compose 构建原生应用
